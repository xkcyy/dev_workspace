# Wiki Schema

## Content Root

- 所有内容页面位于顶层栏目目录
- 推荐目录为 `system/`、`topics/`、`notes/`、`references/`、`maps/`
- 支持 `.md` 和 `.mdx`
- `_site/` 不是内容目录，只是渲染工程

## Frontmatter

```yaml
---
title: 页面标题
summary: 页面摘要
section: system | topics | notes | references | maps
tags:
  - wiki
  - astro
order: 10
draft: false
updatedAt: 2026-04-07
---
```

## Rules

- 一页一个主主题。
- `title` 必填。
- `summary` 用于首页、搜索和列表展示。
- `section` 用于导航分组。
- `tags` 只保留真正可检索的标签。
- 页内链接优先使用相对路径。
- 附件统一放到 `assets/`。
