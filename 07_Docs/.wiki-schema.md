# Wiki Schema

## Content Root

- 所有内容页面位于顶层栏目目录
- 推荐目录为 `system/`、`topics/`、`notes/`、`references/`、`maps/`
- 支持 `.md` 和 `.mdx`
- `_site/` 不是内容目录，只是渲染工程
- 各栏目可通过各自目录下的 `meta.json` 定义分组标题和顺序
- 根目录的 `meta.json` 用于定义一级栏目顺序

## Frontmatter

```yaml
---
title: 页面标题
description: 页面摘要
---
```

## Rules

- 一页一个主主题。
- `title` 必填。
- `description` 用于列表展示和搜索结果摘要。
- 目录分组优先通过文件夹层级和各目录下的 `meta.json` 表达。
- 页内链接优先使用相对路径。
- 附件统一放到 `assets/`。
