# AGENTS.md

本文件定义 `D:\01_DevWorkspace` 这一层的工作区规则。

## 作用范围

- 适用于整个 `D:\01_DevWorkspace` 工作区。
- 如果子目录中存在更深层的 `AGENTS.md`，则以子目录规则为准。

## 当前层定位

- 这里是总工作区根目录，不是单一项目目录。
- 本文件只描述顶层目录设计，不描述具体项目细节。

## 顶层目录

- `01_Work/`：公司或客户相关项目
- `02_OpenSource/`：开源项目
- `03_Personal/`：个人项目
- `04_Labs/`：实验、验证、探索性内容
- `05_Templates/`：模板、脚手架、代码片段
- `06_Tools/`：通用脚本、自动化、环境工具
- `07_Docs/`：文档、笔记、架构材料
- `08_Archive/`：归档内容
- `09_Scratch/`：临时测试和一次性代码

## 简单规则

- `D:\01_DevWorkspace` 本身不要初始化为总 Git 仓库。
- 具体项目应放在合适的子目录中，并各自管理自己的 Git 仓库。
- 当前层只负责分类，不负责细化项目结构。
- 更具体的放置方式和工作规则，由子目录中的 `AGENTS.md` 继续说明。
